# LTM Sampling

An open research project exploring advanced sampling techniques that improve the
training and inference efficiency of large tabular models.

## Research directions

- Adaptive and importance sampling
- Diversity-aware subset selection
- Curriculum and hard-example sampling
- Coreset construction
- Sampling strategies for imbalanced and long-tailed data
- Quality, speed, memory, and cost tradeoffs

## Status

This project is in its initial research and experimentation phase. Benchmarks,
implementations, and reproducible experiment configurations will be added as the
work develops.
